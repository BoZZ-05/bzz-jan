# Clipboard Manager for macOS

一个轻量级、现代化的 macOS 剪贴板管理工具，助你高效管理复制历史。

## ✨ 主要功能

*   **⚡️ 极速唤起**：按下全局热键 `Cmd + Shift + V`，立刻呼出剪贴板历史。
*   **📋 自动记录**：后台静默监听剪贴板变化，自动保存文本记录。
*   **🖱️ 点击即贴**：在列表中点击任一条目，自动将其粘贴到当前活动窗口（无需手动 `Cmd + V`）。
*   **📌 常用置顶**：支持将常用文本置顶，方便随时调用。
*   **☁️ 轻量无感**：采用原生 Swift 开发，无边框悬浮窗设计，占用资源极低。

## 📥 下载与安装

1.  前往 GitHub 仓库的 **Releases** 页面（右侧栏）。
2.  下载最新的 `ClipboardManager.zip` 压缩包。
3.  解压后将 `ClipboardManager.app` 拖入你的 **应用程序 (Applications)** 文件夹。
4.  双击运行即可。

## ⚠️ 首次运行重要说明

由于应用未进行 Apple 开发者签名（这需要每年支付 $99 给 Apple），macOS 默认会拦截从网络下载的它。

### 🔧 推荐：一键修复（彻底解决“无法打开”）
在终端中运行以下命令，即可彻底移除系统的拦截标记：
```bash
sudo xattr -r -d com.apple.quarantine /Applications/ClipboardManager.app
```
*(输入命令后需要输入开机密码，密码输入时不会显示)*

### 备用方案：手动绕过
如果不想用终端：
1.  **右键**点击应用图标，选择 **打开**。
2.  在弹出的警告框中点击 **打开**。
3.  如果提示“文件已损坏”，请必须使用上面的终端命令修复。

### 2. 授予辅助功能权限
为了实现“点击自动粘贴”功能，应用需要控制电脑的按键：
1.  首次运行时，应用会请求 **辅助功能 (Accessibility)** 权限。
2.  如果未弹出或被误关，请前往 **系统设置** -> **隐私与安全性** -> **辅助功能**。
3.  点击 `+` 号，选择 `ClipboardManager` 并确保开关已开启。
4.  **注意**：如果你更新了应用版本，可能需要先在列表中删除旧的 `ClipboardManager`，然后重新添加一次才能生效。

## 🛠️ 本地构建与开发

如果你是开发者，希望修改或自行编译代码：

### 环境要求
*   macOS 13.0+
*   Swift 5.9+ (Xcode 15+)

### 编译步骤
1.  克隆仓库：
    ```bash
    git clone https://github.com/BoZZ-05/bzz-jan.git
    cd bzz-jan
    ```
2.  运行打包脚本（自动编译并生成 DMG）：
    ```bash
    chmod +x package_app.sh
    ./package_app.sh
    ```
3.  生成的应用位于当前目录下。
